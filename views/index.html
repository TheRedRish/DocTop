Express

Server-Side Web Framework

Introduction

Express is a minimal and flexible Node.js web framework that provides a robust set of features for building web and mobile applications. It simplifies server setup and route handling.

Example Code

const express = require('express');
const app = express();

app.get('/', (req, res) => {
  res.send('Hello World!');
});

app.listen(3000, () => console.log('Server running on port 3000'));

Terminal Commands

npm init -y
npm install express
